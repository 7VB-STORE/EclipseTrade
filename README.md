# EclipseTradeBot 🚀

Автоматизированный бот для управления трейдами Steam с удобным графическим интерфейсом и автоматической авторизацией через ChromeDriver.

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Node.js](https://img.shields.io/badge/node-%3E%3D16.0.0-green.svg)
![Electron](https://img.shields.io/badge/electron-latest-blue.svg)

## ✨ Возможности

### 🔹 Управление аккаунтами
- Добавление аккаунтов через форму
- Импорт из `acc.txt` (формат: `login:password:sharedSecret:identitySecret`)
- Импорт из папки с `.maFiles` (Steam Guard файлы)
- Поиск по аккаунтам
- Массовый выбор аккаунтов
- **Пагинация списка** (по 8 аккаунтов на странице)

### 🔹 Работа с инвентарём
- Загрузка инвентаря CS2, TF2, Steam
- Выбор конкретных предметов для отправки
- Выбор количества предметов
- Сортировка: по цене, по названию
- Поиск предметов по названию
- Отображение общей суммы выбранных предметов
- Поддержка валют: RUB, USD, EUR, UAH и др.

### 🔹 Трейды
- Отправка трейдов с выбранных аккаунтов
- Отправка только выбранных предметов
- Автоматическое подтверждение через 2FA
- **Просмотр входящих трейдов**
- **Принятие/отклонение входящих трейдов**
- Защита от Rate Limit

### 🔹 Дополнительно
- Получение 2FA кода
- Автоматическое получение Trade Offer Link
- Копирование ссылки в буфер обмена
- Подробное логирование
- **Открытие профиля Steam в браузере** (ChromeDriver)
- **Автоматическая авторизация** через steam-session
- **Сохранение сессии** для быстрого входа

## 📦 Установка

### Требования
- **Node.js** версии 16.0.0 или выше
- **Google Chrome** (установлен по умолчанию в системе)

### Шаги

1. **Проверьте Node.js:**
```bash
.\check-node.bat
```

2. **Установите Node.js (если не установлен):**
```bash
.\install-node.bat
```

3. **Клонируйте репозиторий:**
```bash
git clone https://github.com/7VB-STORE/EclipsTrade.git
cd EclipsTrade
```

4. **Установите зависимости:**
```bash
npm install
```

5. **Запустите приложение:**
```bash
npm start
```

**Или используйте:**
```bash
.\start.bat
```

## 📖 Использование

### 1️⃣ Добавление аккаунтов

**Через интерфейс:**
1. Нажмите **"+ Добавить"**
2. Введите: Логин, Пароль, Shared Secret, Identity Secret
3. Нажмите **"Добавить"**

**Импорт из acc.txt:**
```
login:password:sharedSecret:identitySecret
```

**Импорт из maFiles:**
- Нажмите **"📁 Папка maFiles"**
- Выберите папку с `.maFile` файлами

### 2️⃣ Получение трейд-ссылки
1. Выберите **один аккаунт**
2. Нажмите **"📋"**
3. Нажмите **"Копировать"**

### 3️⃣ Загрузка инвентаря
1. Выберите аккаунт(ы)
2. Выберите инвентарь (CS2, TF2, Steam)
3. Выберите валюту
4. Нажмите **"Загрузить"**

### 4️⃣ Выбор предметов
- Используйте **поиск** и **сортировку**
- Выберите количество предметов
- Общая сумма показывается автоматически

### 5️⃣ Отправка трейда
1. Вставьте **Trade Offer Link**
2. Выберите аккаунты и предметы
3. Нажмите **"📤 Отправить трейд"**

### 6️⃣ Входящие трейды
1. Выберите аккаунт
2. Нажмите **"📥 Входящие трейды"**
3. Нажмите **"✓ Принять"** или **"✗ Отклонить"**

### 7️⃣ 2FA код
1. Выберите аккаунт
2. Нажмите **"🔐"**
3. Скопируйте код

### 8️⃣ Открытие профиля Steam
1. Выберите **один аккаунт**
2. Нажмите **"🌐 Steam"** (или аналогичную кнопку)
3. Браузер откроется с авторизованным аккаунтом
4. **Повторный вход быстрый** — сессия сохраняется!

### 🔐 Хранение сессий

**Сессии сохраняются автоматически в профиле Chrome!**

**Расположение:** `%TEMP%\eclips-chrome\`
- Открывается через ChromeDriver с флагом `--user-data-dir`
- Chrome сам сохраняет cookies, localStorage, sessionStorage
- **При повторном запуске** сессия загружается автоматически

**Преимущества:**
- ✅ Не нужно управлять файлами сессий вручную
- ✅ Chrome сам обновляет cookies
- ✅ Быстрая загрузка (не нужно авторизоваться каждый раз)
- ✅ Автоматическая очистка при перезагрузке ПК (temp папка)

**Важно:**
- 🔒 Папка во временной директории — защищена системой
- 🗑️ Очищается при очистке Temp или перезагрузке
- 🔄 При очистке придётся авторизоваться заново

**Временные файлы:**
- `chrome-profiles/` — находится в `%TEMP%\eclips-chrome\`
- Содержит: cookies, кэш, localStorage, sessionStorage

## 🔒 Безопасность

⚠️ **Важно:**
1. Не передавайте `accounts.json` третьим лицам
2. Храните секреты в безопасном месте
3. Не используйте на основных аккаунтах
4. Делайте резервные копии `accounts.json`
5. Сессии хранятся в `%TEMP%\eclips-chrome\` — Chrome управляет ими автоматически

### 📍 Расположение важных файлов

| Файл | Где находится | Что содержит |
|------|--------------|--------------|
| `accounts.json` | `EclipsTrade/` | Логины, пароли, 2FA секреты |
| `chrome-profiles/` | `%TEMP%\eclips-chrome\` | Cookies сессий, кэш браузера |
| `prices_cache.json` | `EclipsTrade/` | Кэш цен предметов |

## 🐛 Решение проблем

| Ошибка | Решение |
|--------|---------|
| Неверная ссылка на трейд | Проверьте наличие `steamcommunity.com` и `tradeoffer` |
| Нужен sharedSecret | Добавьте в настройках аккаунта |
| Rate Limit | Подождите 2-5 минут |
| Throttle | Подождите 5-30 минут |
| ChromeDriver error | Обновите Chrome или перезапустите приложение |
| Аккаунт не авторизуется | Проверьте логин/пароль и 2FA |
| Браузер не открывается | Проверьте, что Chrome установлен |
